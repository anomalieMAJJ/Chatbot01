## UQAM{fl4g1fy}

Implémentez un chatbot CLI qui, en recevant une chaîne de caractères en entrée, convertit cette chaîne au format de flag CTF AGEEI et la renvoie à l'utilisateur.

Exemple:
```
> touch grass.c
UQAM{t0uch_gr455.c}
```
